# Game Sound Files

This directory should contain the following sound files in **MP3 or OGG format**:

## Required Sound Files

### 1. `wheel_spin.mp3` (or .ogg)
- **Description**: Continuous wheel spinning sound
- **Duration**: 2-3 seconds (will be looped)
- **Suggested Sound**: Ratchet clicking sound, game show wheel spinning
- **Usage**: Played in loop while wheel is spinning
- **Source Suggestions**:
  - Freesound.org: Search "wheel spin", "ratchet", "game show wheel"
  - Pixabay: Free game sound effects

### 2. `correct_answer.mp3` (or .ogg)
- **Description**: Positive feedback for correct letter
- **Duration**: 0.5-1 second
- **Suggested Sound**: Bell ding, success chime, positive beep
- **Usage**: Played when player guesses correct letter
- **Source Suggestions**:
  - Freesound.org: Search "success", "correct", "bell ding"
  - Pixabay: Success sound effects

### 3. `incorrect_answer.mp3` (or .ogg)
- **Description**: Negative feedback for wrong letter
- **Duration**: 0.5-1 second
- **Suggested Sound**: Buzzer, wrong answer sound, descending tone
- **Usage**: Played when player guesses wrong letter
- **Source Suggestions**:
  - Freesound.org: Search "buzzer", "wrong", "error"
  - Pixabay: Error sound effects

### 4. `bankrupt.mp3` (or .ogg)
- **Description**: Dramatic negative sound for bankruptcy
- **Duration**: 1-2 seconds
- **Suggested Sound**: Losing horn, sad trombone, dramatic negative chord
- **Usage**: Played when player lands on Bankrupt slice
- **Source Suggestions**:
  - Freesound.org: Search "sad trombone", "fail", "game over"
  - Pixabay: Fail sound effects

### 5. `extra_turn.mp3` (or .ogg)
- **Description**: Positive jingle for extra turn
- **Duration**: 1-2 seconds
- **Suggested Sound**: Ascending chime, fanfare, power-up sound
- **Usage**: Played when player lands on Extra Turn slice
- **Source Suggestions**:
  - Freesound.org: Search "power up", "bonus", "fanfare"
  - Pixabay: Bonus sound effects

## Free Sound Resources

1. **Freesound.org** - Free sound effects with Creative Commons licenses

3. **Zapsplat** - Free sound effects for game development
4. **OpenGameArt.org** - Free game audio assets

## File Format Notes

- Preferred format: **MP3** (widely supported, good compression)
- Alternative: **OGG** (open format, good quality)
- File size: Keep under 100KB per file for performance
- Sample rate: 44.1 kHz recommended
- Bit rate: 128-192 kbps for MP3

## License Requirements

Ensure all sound files are either:
- Public domain
- Creative Commons (with proper attribution if required)
- Royalty-free with commercial use license

Document the source and license for each sound file.
